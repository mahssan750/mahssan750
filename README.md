## 🚀 Flagship Project — Customer Churn Prediction

> **Identify at-risk customers before they leave** — an ML system on real e-commerce data that answers one question: *"Will this customer stop buying in the next 90 days?"*

Retaining a customer costs **5–7× less** than acquiring a new one. Every customer is scored **0–100%** and ranked by urgency, so the retention team knows exactly **who to call, who to email, and who to leave alone**.

**Pipeline:** behavioural feature engineering → train &amp; compare **Logistic Regression · Random Forest · XGBoost** (Optuna tuning) → score + risk-tier each customer → serve via **FastAPI** with scheduled daily retraining and a `pytest` suite.
